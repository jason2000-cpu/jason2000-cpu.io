Movie Pulse

  • A Movie Search Application: React, Tailwind CSS, MongoDB, NodeJs, Express Js Gemini AI, TMDB API
  • Website Link: link

Movie Pulse is a cutting-edge movie platform built with the MERN stack (MongoDB, Express, React, Node.js), designed for movie enthusiasts to explore and discover films effortlessly. With integration of the TMDB API, it offers access to a vast library of movie data, including the latest releases, classics, and trending titles. Combined with Gemini AI, Movie Pulse provides a seamless, context-aware search experience, helping users find the perfect movie recommendations based on their preferences
